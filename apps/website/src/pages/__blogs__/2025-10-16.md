## 功能

1. 支持解析自定义的 `<code-example src="xxx.md" />`，并渲染对应的代码示例
2. 支持解析自定义的 `<props-table src="xxx.md" />`，并渲染对应的 API 文档
3. 支持解析自定义的 `<code-example src="xxx.md" />`，并渲染对应的组件

## Vite 插件实现

### 自定义 `resolveId` 函数

如果是符合要求的文件，将会生成一个唯一的 ID，并返回。

```ts
{
  resolveId: (sourceId, importer) => {
    if (isDocumentFile(sourceId)) {
      return getUniqueId(sourceId)
    }
  }
}
```

### 自定义 `load` 函数

如果检测到在 `resolveId` 中生成的唯一 ID 时，将会读取文件内容，并返回解析后的内容。

```ts
{
  load: (sourceId) => {
    if (isUniqueId(sourceId)) {
      const content = readFileSync(sourceId, 'utf-8')
      return parseDocument(content)
    }
  }
}
```
